6.4.2 Coding style

A natural restriction on convertible code is that it should be written in MyHDL style: cooperating generators, communicating through signals, and with sensitivity lists specifying wait points and resume conditions. Supported resume conditions are a signal edge, a signal change, or a tuple of such conditions.
